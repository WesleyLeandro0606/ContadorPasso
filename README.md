# 📱 Contador de Passos – APP SPORT

Aplicativo Android desenvolvido em Java no Android Studio.

O objetivo do app é calcular a distância percorrida com base na quantidade de passos informada pelo usuário.

---

# 🚀 Funcionalidades

✅ Inserir quantidade de passos

✅ Selecionar tamanho do passo:
- Passo curto = 0,5m
- Passo médio = 0,7m
- Passo longo = 1m

✅ Opção "Estava correndo"
- aumenta a distância em 10%

✅ Exibe a distância percorrida

✅ Interface intuitiva

✅ Imagem personalizada

✅ Fundo colorido

---

---

# 📐 Fórmula do Cálculo

Distância:

passos × tamanho do passo

Caso esteja correndo:

distância + 10%
